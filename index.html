Messaging system

Messaging system as a communicating infrastructure is a safety-critical component of autonomous vehicles. An autonomous vehicle communication system (AVCS) is a shared-memory bus, which is a messaging infrastructure to allow a variety of subsystems to communicate through shared message buffers.

PiCore Framework

PiCore is an event-based and parametric rely-guarantee framework for concurrent reactive systems (e.g., concurrent OS kernels, control systems, communication systems). We create parameterized axiomatic model for messaging systems by extending the PiCore framework.

Axiomatic Model of Messaging System and Formal specification and verification of the DGPS system of UISEE Autonomous driving systems

We develop an axiomatic model for messaging systems. The model contains a set of axioms to constrain the architecture, its configuration, and rely-guarantee conditions of messaging systems. The model supports dynamic configuration of message buffers in the data bus. Then, we instantiate the axiomatic model to create the formal specification of the DGPS system and use the rely-guarantee proof system in PiCore to verify the functional correctness and invariant preservation.
